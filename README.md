### **Description**

This project is a C++ program that loads a noisy audio file, processes it using an ONNX-based deep learning model to perform speech enhancement (like denoising), and saves the cleaned audio back to a .wav file.


The code performs the following key steps:

1. Initialize an ONNX model for audio enhancement.

2. Load an input noisy .wav file into a tensor.

3. Transform the audio (likely FFT) for model processing.

4. Split audio into chunks and run inference per chunk.

5. Concatenate output chunks and inverse transform.

6. Save the enhanced audio to a new .wav file.
